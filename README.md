# COL Prayer Course App

A modern, interactive web application for the 10-week COL Intercessory Prayer Course.

## Features

- **Course Progress Tracking**: Track your progress through all 10 weeks
- **Daily Lessons**: Access daily prayer lessons and study materials
- **Prayer Journal**: Record personal reflections and prayer experiences
- **Note Taking**: Take notes on each lesson for later reference
- **User Dashboard**: View your progress at a glance

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the App

```bash
npm run dev
```

The app will open automatically at `http://localhost:5173`

### Building for Production

```bash
npm run build
```
